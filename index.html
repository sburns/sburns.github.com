Projects

PyCap

I designed and maintain a python package for communicating with REDCap projects through the API, PyCap. It provides an easy way for external python applications to export and import data to and from REDCap projects. It is the basis for much of what I've managed to do at EBRL.

Red-Light

The main REDCap web-app is good for the most part but doing ad-hoc searches against projects is terrible. I wrote a small web-app to expedite this process. It works against the REDCap API. There's a live instance at Heroku and you can view the code on github.
